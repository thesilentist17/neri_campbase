# Журнал змін (Changelog)

Усі помітні зміни в цьому проєкті будуть документуватися в цьому файлі.

## [0.1.1] — 2026-06-16

### Додано
- Нова прихована сторінка модератора (`/admin-panel`) для перевірки та публікації запропонованих ігор.
- Система авторизації через секретний пароль для адмін-панелі.
- Безпечне збереження пароля через змінні оточення `NEXT_PUBLIC_ADMIN_PASSWORD` на Vercel.
- Нова сторінка-візитка для Школи Аніматора (`/school`) з контактами та посиланнями.
- Кнопка «Поділитися грою» на сторінці активності, яка копіює посилання в буфер обміну з візуальним підтвердженням «Скопійовано!».
- Додаткові інформаційні значки на картках активностей у категоріях: «Підготовка», «Учасники», «Аніматори» та статус реквізиту.
- **Інтеграція з Google Drive API (OAuth2) для автоматичного завантаження прикріплених до ігор документів прямо у хмарне сховище (економія місця в базі даних).**
- **Власне красиве модальне вікно для введення пароля модератора на сторінці гри (замість застарілого `window.prompt`).**
- **Відображення клікабельних посилань на прикріплені матеріали (фото та документи) прямо в Адмін-панелі під час модерації.**

### Змінено
- Фільтри пошуку перероблені на зручніші поодинокі поля замість складних діапазонів «від/до».
- Логіка фільтра аніматорів тепер порівнює наявну кількість із мінімально необхідною за базою даних.
- Замінено чекбокс реквізиту на трьохпозиційний список (Не має значення / Тільки з реквізитом / Без реквізиту).
- Метадані сайту: змінено стандартний заголовок "Create Next App" на власну назву.
- **Логіка завантаження файлів у формі пропозиції: картинки йдуть у Supabase Storage, а важкі документи (PDF, DOCX) — на Google Диск.**
- **Розділення статусів модерації: нові ігри отримують статус `pending` (жовта плашка), а ігри, відправлені на редагування після публікації — `editing` (фіолетова плашка).**

## [0.1.0] — 2026-06-15

### Додано
- Базовий запуск проєкту на Next.js та Tailwind CSS.
- Підключення бази даних Supabase та налаштування таблиці `activities`.
- Головна сторінка з категоріями ігор.
- Сторінка категорії з виведенням списку опублікованих активностей.
- Детальна сторінка активності з технічними картками параметрів.
- Форма «Запропонувати активність» з інтерактивною підказкою щодо структури тексту.
- Перший успішний деплой проєкту на Vercel.
